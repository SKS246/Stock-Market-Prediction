Stock Price Predictor

About The Product

As seen from the title, this product is about the stock market. This product in brief is a stock market predictor which analyzes previous trends in the market to predict the stock market price for the next few days.

NIFTY 50: The website predicts the stock prices of all the stocks in the NIFTY 50 Index. The NIFTY 50 is a benchmark Indian stock market index that represents the weighted average of 50 of the largest Indian companies listed on the National Stock Exchange.

ML Model: The website queries data from a complex machine learning algorithm which is trained using the LSTM ( Long Short Term Memory ) algorithm to analyze the previous trends and predict the price of a specific stock for the next few days.

Dataset: The same data from the model then is imported to the database of the website. The website uses the http POST method to extract data from the database and display the predictions for you - the user on the website.
